# E-commerce App

## Description

Simple e-commerce application built with:

- Angular (frontend)
- .NET Web API (backend)
- SQL Server (to be added)

## Features (so far)

- Backend API running
- Angular frontend running
- Connection between frontend and backend

## How to run

### Backend

cd backend
dotnet run

### Frontend

cd frontend
ng serve

Open http://localhost:4200

This is a work in progress project.
